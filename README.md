## Why this exists

Spotify Wrapped shows your music habits in a fun way.  
So the obvious question was: if we can wrap music, why not wrap AI?

AI Wrapped turns your ChatGPT history into a visual story of your prompts, habits and deep dives.

---

## Features

- Upload your `conversations.json` from ChatGPT export  
- Wrapped style slides that tell your AI usage story  
- Stats for:
  - Total prompts  
  - Total conversations  
  - Active days  
  - Words typed  
  - Longest conversation  
  - Most active weekday and hour  
- Dashboard with charts (weekday and hour activity)  
- Keyboard navigation for slides  
- 100% local processing in the browser

## How to run locally

```bash
# clone the repo
git clone https://github.com/viratnigam18/ai_wrapped.git
cd ai_wrapped

# open index.html directly OR start a local server
```

If you want a simple local server:

```bash
python -m http.server 8000
# then open http://localhost:8000
```
